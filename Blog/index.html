Blog sobre html

¿Qué es html?

HTML (Lenguaje de marcado de hipertexto o HyperText Markup Language por sus siglas en inglés) es un lenguaje descriptivo que especifica la estructura de las páginas web.

¿Para qué sirve?

se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

¿Cómo funciona html para mostrar contendio web?

Es el código que hace funcionar Internet. Es un lenguaje de marcado, lo que significa que utiliza etiquetas para definir elementos, como marcar texto en negrita o mostrar imágenes con código.
